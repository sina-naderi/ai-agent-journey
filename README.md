## 🚀 Setup

```bash
git clone https://github.com/sina-naderi/ai-agent-journey.git
cd ai-agent-journey
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Create a `.env` file in the root directory:

```
ANTHROPIC_API_KEY=your_key_here
MY_NAME=YourName
```
